# LEBU - 부산 관광 정보 웹사이트

포스코 코딩온 웹 개발자 부트캠프 9기 프로젝트 1

## 📋 프로젝트 소개

LEBU는 부산의 주요 관광지와 맛집 정보를 제공하는 웹사이트입니다. 실시간 날씨 정보, 인터랙티브 지도, 다국어 지원 등 다양한 기능을 통해 부산을 방문하는 관광객들에게 유용한 정보를 제공합니다.

## ✨ 주요 기능

### 1. 관광지 정보 제공
- **광안리**: 바다, 야경, 써핑 명소
- **다대포**: 노을맛집, 포장마차, 음악분수
- **송도해상케이블카**: 해변길, 데이트코스, 인생사진
- **청사포 해변열차**: 해변길을 따라가는 이색 경험
- **감천문화마을**: 부산의 대표 문화마을

### 2. 실시간 날씨 정보
- 기상청 초단기예보 API를 활용한 실시간 날씨 표시
- 날씨 아이콘과 온도 정보 제공
- 부산 지역 날씨 정보

### 3. 인터랙티브 지도
- 부산 지역 지도 이미지 맵
- 클릭 가능한 관광지 영역
- 반응형 이미지 맵 지원

### 4. 맛집 정보
- 지역별 추천 맛집 리스트
- 상세 정보 모달 (메뉴, 위치, 소개)
- Google Maps 연동

### 5. 전시회/행사 정보
- 부산 지역 전시회 및 행사 정보 슬라이더
- 외부 링크 연결

### 6. 다국어 지원
- 한국어 (기본)
- 영어 (English)
- 중국어 (中文)

## 🛠 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 애니메이션
- **JavaScript (Vanilla JS)**: 동적 기능 구현
- **jQuery**: DOM 조작 및 이벤트 처리
- **jQuery UI**: UI 컴포넌트
- **jQuery rwdImageMaps**: 반응형 이미지 맵

### 외부 API 및 서비스
- **기상청 초단기예보 API**: 날씨 정보
- **Google Maps Embed API**: 지도 표시
- **SCoreDream 폰트**: 웹폰트
- **Material Symbols**: 아이콘

## 🚀 설치 및 실행 방법

### 1. 저장소 클론
```bash
git clone [repository-url]
cd project1
```

### 2. 로컬 서버 실행
프로젝트는 정적 파일로 구성되어 있으므로 로컬 서버가 필요합니다.

#### Python 사용 시:
```bash
cd web
python -m http.server 8000
```

#### Node.js 사용 시:
```bash
npx http-server web -p 8000
```

#### VS Code Live Server 확장 사용:
- VS Code에서 `web` 폴더를 열고
- `index.html` 파일을 우클릭하여 "Open with Live Server" 선택

### 3. 브라우저에서 접속
```
http://localhost:8000
```

## 🌐 다국어 지원

URL 파라미터를 통해 언어를 변경할 수 있습니다:
- 한국어: `index.html` (기본)
- 영어: `index.html?en`
- 중국어: `index.html?cn`

각 정보 페이지도 `cn/`, `en/` 폴더에 다국어 버전이 제공됩니다.

## 🔑 API 키 설정

### 날씨 API
`web/js/weather.js` 파일에서 기상청 API 서비스 키를 설정해야 합니다.

```javascript
var queryParams = '?' + encodeURIComponent('serviceKey') + '='+'YOUR_API_KEY';
```

**참고**: 현재 코드에 하드코딩된 API 키가 포함되어 있습니다. 실제 배포 시 환경 변수나 설정 파일로 관리하는 것을 권장합니다.
