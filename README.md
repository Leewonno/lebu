# 부산 관광 정보 웹사이트, LEBU

## 📋 프로젝트 개요

LEBU는 부산의 주요 관광지와 맛집 정보를 제공하는 웹사이트입니다. 실시간 날씨 정보, 다국어 지원을 통해 부산을 방문하는 관광객들에게 유용한 정보를 제공합니다.

[🔗 배포 링크](https://poetic-druid-deac2a.netlify.app/)

<br />

## ⚡️ 주요 기능
- 기상청 초단기예보 API를 활용한 실시간 부산 지역 날씨를 제공합니다.
- 부산 관광지, 맛집, 행사 정보를 제공합니다.
- 한국어 외에도 다국어(중국어, 영어)를 지원합니다.
- Netlify를 이용해 배포하였습니다.
